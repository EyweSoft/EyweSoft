ABOUT LibreOffice.ORG PORTABLE
=============================
The LibreOffice Portable Launcher allows you to run LibreOffice.org from a removable drive whose letter changes as you move it to another computer.  The program can be entirely self-contained on the drive and then used on any Windows computer.

INSTALLATION / DIRECTORY STRUCTURE
==================================
By default, the program expects these directory structure:

-\ <--- Directory with LibreOfficePortable.exe
	+\App\
		+\AppInfo\
				+\Launcher\
		+\LibreOffice\
		+\java\
	+\Data\
		+\settings\


LibreOfficePortable.ini CONFIGURATION
====================================
The LibreOffice Portable Launcher ist actualy based on the PortableApps.com Launcher. 
There is normaly no need for changing LibreOfficePortable.ini in the App\AppInfo\Launcher directory.
I You want to know how the parameters work, take a look at http://portableapps.com/manuals/PortableApps.comLauncher/

LibreOfficePortableSettings.ini CONFIGURATION
====================================
There are some paramters You could change in this ini in the directory Data\settings
