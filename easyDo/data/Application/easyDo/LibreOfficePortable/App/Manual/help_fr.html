La suite bureautique portable

LibreOffice.org Portable est une suite bureautique portable complète -- traitement de texte, tableur, diaporamas, dessins, etc. -- empaquetée ici afin d'en faire une application portable. Vous pourrez alors travailler avec tous vos documents sur un iPod, une clé USB, un disque dur externe ou tout autre média amovible. Elle a exactement les mêmes fonctionnalités que LibreOffice.org, dont : le support de nombreux formats de fichiers (Microsoft Word, Word Perfect, Microsoft Excel, etc), l'export PDF, la correction orthographique en temps réél et plus encore... le tout dans une interface ou vous retrouverez facilement vos marques. Enfin, elle ne laisse pas d'informations personnelles sur la machine sur laquelle vous exécutez le logiciel, vous pourrez donc emporter votre suite bureautique avec vous où que vous alliez. Plus d'informations LibreOffice.org...
