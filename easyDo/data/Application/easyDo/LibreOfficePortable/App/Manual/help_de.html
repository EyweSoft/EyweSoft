LibreOffice Portable Hilfe

das komplette Office-Paket zum Mitnehmen

LibreOffice Portable ist das komplette LibreOffice Office-Paket -- Textverarbeitung, Tabellenkalkulation, Präsentationen erstellen, Zeichenprogramm, Datenbank -- zusammengepackt mit einem Startprogramm von  PortableApps.com als eine Portable Applikation (portable app), so daß Sie mit ihren Daten und Dokumenten auf einem USB-Stick, einer mobilen Festplatte, einem MP3-Player oder jedem anderen mobilen Speicher arbeiten können. Es hat all die gleichen großartigen Möglichkeiten von LibreOffice, wie zum Beispiel die Unterstützung verschiedener Dateiformate (Microsoft Word, Word Perfect, Microsoft Excel, etc), Ausgabe als PDF, eine Rechtschreibprüfung bei der Eingabe und vieles mehr ... alles unter einer vertrauten Oberfläche. Zusätzlich hinterlässt es keine Informationen auf der Maschine, auf der Sie damit arbeiten, So daß Sie Ihr gesamtes Büro mitnehmen können, wohin Sie auch gehen. Erfahren Sie mehr über LibreOffice...
